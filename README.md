# Tailoring Management System

A modern full-stack tailoring management application built with **Angular 21**, **PrimeNG**, **Tailwind CSS**, **Express.js**, **TypeScript**, and **MongoDB**. The system is designed to streamline tailoring operations by providing an intuitive interface for managing customer measurements, bookings, stitching details, delivery schedules, expenses, and payment records.

## Features

* Customer registration and profile management
* Measurement management for Kameez and Shalwar
* Booking and delivery date tracking
* Stitching customization and tailoring preferences
* Expense and payment management
* Balance and advance payment tracking
* Responsive and modern UI with PrimeNG and Tailwind CSS
* Form validation and error handling
* RESTful API architecture
* MongoDB database integration
* Type-safe development using TypeScript
* Scalable and maintainable code structure
